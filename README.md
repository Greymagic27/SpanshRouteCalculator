# Spansh Route Calculator
This tool allows you to get the most efficient route using the spansh neutron plotter.

To use the tool, download the latest release and then run it using Java 25.

The tool will then prompt you to enter details such as the system you are currently in, where you want to jump to, your jump range and if you are using overcharge supercharge (for the Caspian Explorer)
<img width="367" height="111" alt="image" src="https://github.com/user-attachments/assets/ffd3919a-07fd-4d65-9fb8-84a3f6ba8773" />

Once these details are entered, the tool will begin to search through the neutron plotter efficiency ranges to find the one resulting in the least jumps needed.
When the tool has finished, it will produce a URL to the best route, so you can follow it via the spansh website.
<img width="1714" height="980" alt="image" src="https://github.com/user-attachments/assets/4af3c409-3338-49b4-b28a-78da9023f134" />

This tool will also remember the last information provided to it, as it generates a .properties file in the directory you ran it in last.
